Advanced Keyboard Activity Monitor

This project is an advanced and ethical Keyboard Activity Monitoring System developed using Python as part of the Prodigy InfoTech Cyber Security Internship.

The application monitors and analyzes keyboard activity ONLY inside the application window for educational and cybersecurity learning purposes.

Features
Real-Time Keyboard Event Monitoring
Keyboard Activity Analytics
Key Frequency Tracking
Session Statistics
TXT Log Generation
JSON Export Support
CSV Export Support
Save & Clear Logs
Interactive GUI Interface
Ethical and Secure Monitoring
